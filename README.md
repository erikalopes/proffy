# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/erikalopes/proffy.git
```
### üì¶ Run API

```bash
# Go to server folder
$ cd proffy/server

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Access API at http://localhost:3333/

### üíª Run Web Project

```bash
# Go to web folder
$ cd proffy/web

# Install Dependencies
$ yarn install

# Run Aplication
$ yarn start
```
Go to http://localhost:3000/ to see the result.
